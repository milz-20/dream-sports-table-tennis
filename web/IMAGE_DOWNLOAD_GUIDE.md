# Image Download Guide for Dream Sports Table Tennis

## Quick Start - Download Free Stock Photos

### 1. Hero Images (Homepage)
**Search: "table tennis player action"**
- Visit: https://unsplash.com/s/photos/table-tennis
- Download a high-quality action shot (1920x1080 or larger)
- Save as: `hero-table-tennis.jpg`

### 2. Coaching Hero
**Search: "table tennis coach training"**
- Visit: https://unsplash.com/s/photos/table-tennis-coach
- Download coaching/training image (1600x900 or larger)
- Save as: `coaching-hero.jpg`

### 3. Equipment Hero
**Search: "table tennis equipment paddle"**
- Visit: https://unsplash.com/s/photos/table-tennis-equipment
- Download equipment showcase (1600x900 or larger)
- Save as: `equipment-hero.jpg`

### 4. Product Category Images

### 5. Testimonial Profile Images
**Search: "professional headshot" or use portrait photos**
- Download 3 diverse profile photos (square, 400x400+)

  - `testimonial-sarah.jpg`

## Image Optimization (Before Upload):

Use these free tools to compress images:
- **TinyPNG**: https://tinypng.com/ (Compress JPG/PNG)
- **Squoosh**: https://squoosh.app/ (Advanced compression)
- **ImageOptim**: https://imageoptim.com/ (Desktop app for Mac)

**Target file sizes:**
- Hero images: < 500KB
- Product images: < 300KB
- Thumbnails/profiles: < 100KB
