## Recommended Sources for Stock Photos

### Free Stock Photo Sites:
- **Unsplash** (https://unsplash.com/) - Search: "table tennis", "ping pong", "sports equipment"
- **Pexels** (https://pexels.com/) - High-quality free images
- **Pixabay** (https://pixabay.com/) - Free images and videos

### Search Keywords:
- "table tennis player"
- "ping pong match"
- "table tennis coaching"
- "table tennis equipment"
- "ping pong paddle"
- "table tennis ball"
- "sports training"
- "table tennis tournament"

## How to Add Images:

1. Download images from stock photo sites
2. Rename them according to this guide
3. Place them in `web/src/assets/images/` folder
4. Images will be automatically imported in the components

## Optimization Tips:
- Use JPG for photos (smaller file size)
- Use PNG for logos with transparency
- Recommended max file size: 500KB per image
- Use online tools like TinyPNG to compress images
